# Multi-Domain Data Analysis Portfolio

**Lekhasri** | Vijayawada, Andhra Pradesh | March 2026

A collection of **5 complete end-to-end data analysis projects** across Retail, Education, Weather, Healthcare, and Finance — built with Python (pandas, matplotlib, seaborn).

### Quick Start – Download Everything at Once

1. Click the green **Code** button (top-right corner)  
2. Click **Download ZIP**  
3. Extract the ZIP file on your computer  
4. Open any .ipynb file in Jupyter Notebook, Google Colab, or VS Code  
5. Check **reports/** folder for detailed Word documents (.docx) with insights & recommendations

### Projects Overview

1. **Retail – Supermarket Sales Analysis**  
   Domain: Retail  
   Focus: Daily/monthly sales trends, top products, customer behavior

2. **Education – Student Performance Analysis**  
   Domain: Education  
   Focus: Pass/fail rates, subject averages, attendance correlation

3. **Weather – Temperature Trends Analysis**  
   Domain: Weather  
   Focus: Monthly/seasonal patterns, extreme hot/cold days

4. **Healthcare – Hospital Operations Analysis**  
   Domain: Healthcare  
   Focus: Length of Stay (LOS), revenue by case type/specialty, early discharge rate

5. **Finance – House Prices Analysis**  
   Domain: Finance / Real Estate  
   Focus: Price drivers (area, bedrooms, location, type, age), correlations
